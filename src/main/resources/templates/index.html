SDG-OD linker

SDG-OD is a Web application that was born from the following assumption: If there are so many data sets around the world that contain information from a wide range of subjects, there must be some of them that can help measure and achieve the Sustainable Development Goals (SDG) proposed by the United Nations.

The purpose of this application is to provide a platform in which anyone can find data related to many of the Sustainable Development Goals easily. Additionally, there's helpful information generated by the application that helps understand it's purpose further.

This work leverages technologies provided by The Semantic Web. With the purpose of achieving our goal of connecting Open Data and the Sustainable Development Goals, we established the following architecture and implemented it within this application.

Our methodology consists of four separate stages. Extracting Open Data from public access data portals, transforming such data, semantically annotate the Sustainable Development Goals to link them with related data sets, and visualizing the results.

Open Data Extraction

In this first stage, the application extracts data sets from public access data portals that use the CKAN platform. Doing so helps to maintain a standardized extraction method across many different sources. The application then stores some of the data locally to use it in the next stage.

Data transformation

This stage does some pre-processing to clean the extracted data sets and homogenize them as much as possible. Afterward, the application uses the semantic vocabulary DCAT to create a knowledge representation from the data sets and upload a graph into a triplestore.

SDG semantic annotation

Because the data from each Sustainable Development Goal is not machine-readable, the application uses Natural Language Processing to obtain a meaningful representation of each goal, target, and indicator. This process was accomplished by using the tool FRED that creates a formal semantic graph from natural language. The resulting graphs from FRED are then transformed into a concept taxonomy using the SKOS vocabulary. Finally, the application uses the previously transformed data sets along with the concept taxonomy from each SDG to discover links between them.

OD and SDG Visualizer

At this point, the application has created a semantic representation of the extracted data sets, the Sustainable Development Goals, and relationships between them. This web page offers different ways to visualize such data. Every data set can be seen and queried using the data sets browser. After accessing a particular data set, the application will display if it has any relationship with any of the 17 SDG. There is another browser that displays only the data sets that have relationships with one or more goals. Information about the SDG representation can be found on this page as well.
